Weather Data Analysis:

 Overview:
This project analyzes a synthetic weather dataset for the year 2023 using Python.
 Tools:

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

 Data:

Columns used:

* Date
* Temperature
* Rainfall
* Humidity
* Condition
* Month
* Season

 Analysis:

* Temperature trend (line chart)
* Rainfall by season (bar chart)
* Temperature distribution (histogram)
* Temperature vs Humidity (scatter plot)
* Weather conditions (pie chart)

How to Run:

```bash
pip install pandas numpy matplotlib seaborn
python weather_analysis.py
```
